# Job Market Analysis - Data analytics Project

## Overview
This project analyzes job market trends using real-world job posting data. 
It focuses on understanding salary patterns, industry demand, education impact, job roles, and required skills.
The analysis is performed using SQL, Python, and Power BI to provide useful insights for job seekers and employers.

## Tools Used
SQL
Excel
Power BI
Python (JUpyter Notebook)

## Files in this Repository
Jon market analysis (PRDA-04).pptx-project presentation
job_market_analysis.ipynb - python analysis using Jupyter notebook
Job_Market_Analysis_dashboard.pbix – Power BI dashboard
job_market_data_cleaning.sql - sql data cleaning queries
job_market_data1.csv – Dataset 1
job_market_data2.csv – Dataset 2

## key insights
IT and Analytics industries offer higher salaries.
Higher education leads to better salary packages.
Python, SQL, and Cloud skills are highly demanded.
Data Scientist and ML Engineer roles have maximum openings.
Multi-skilled candidates get better career opportunities.
